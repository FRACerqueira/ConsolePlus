The **Component** group contains **9** emoji &mdash; skin-tone and hair modifiers used to build other emoji.

Use each entry as a `:shortcode:` inside [markup](../markup.md#emoji), or as a strongly-typed
`Emoji` constant. You can also group-qualify a shortcode as `:component/shortcode:`. See the
[Emoji guide](../emoji.md#group-qualified-shortcodes) for lookup and normalization rules.

|:-----:|----------|-----------|

| 🦱 | `Emoji.CurlyHair` | `:curly_hair:` |

## EmojiTravelAndPlaces Class

Strongly\-typed constants for the official Unicode emoji group "TravelAndPlaces"\.
This mirrors the group\-qualified shortcode form \(for example
`:travelandplaces/name:`\) with compile\-time safety and IntelliSense\.

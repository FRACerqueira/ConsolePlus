## EmojiAnimalsAndNature Class

Strongly\-typed constants for the official Unicode emoji group "AnimalsAndNature"\.
This mirrors the group\-qualified shortcode form \(for example
`:animalsandnature/name:`\) with compile\-time safety and IntelliSense\.

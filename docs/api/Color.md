## Color\.AdjustForegroundColorForContrast\(Color, double\) Method

Adjusts the foreground color to ensure adequate contrast with the background color, only if necessary\.

```csharp
public ConsolePlusLibrary.Color AdjustForegroundColorForContrast(ConsolePlusLibrary.Color backgroundColor, double minimumContrastRatio=2.5);
```
#### Parameters

<a name='ConsolePlusLibrary.Color.AdjustForegroundColorForContrast(ConsolePlusLibrary.Color,double).backgroundColor'></a>

`backgroundColor` [Color](Color.md 'ConsolePlusLibrary\.Color')

The background color\.

<a name='ConsolePlusLibrary.Color.AdjustForegroundColorForContrast(ConsolePlusLibrary.Color,double).minimumContrastRatio'></a>

`minimumContrastRatio` [System\.Double](https://learn.microsoft.com/en-us/dotnet/api/system.double 'System\.Double')

The minimum acceptable contrast ratio \(default: 2\.5\)\. Common values: 4\.5 \(AA\), 7 \(AAA\)\.

#### Returns
[Color](Color.md 'ConsolePlusLibrary\.Color')  
The adjusted foreground color if contrast is inadequate, otherwise the original color\.

### Remarks
This method evaluates the contrast ratio between the current color and the background\.
If the contrast is less than the minimum required, it returns either white or black, whichever provides better contrast\.
If the contrast is already adequate, the original color is returned unchanged\.

## EmojiFoodAndDrink Class

Strongly\-typed constants for the official Unicode emoji group "FoodAndDrink"\.
This mirrors the group\-qualified shortcode form \(for example
`:foodanddrink/name:`\) with compile\-time safety and IntelliSense\.

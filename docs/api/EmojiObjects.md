## EmojiObjects Class

Strongly\-typed constants for the official Unicode emoji group "Objects"\.
This mirrors the group\-qualified shortcode form \(for example
`:objects/name:`\) with compile\-time safety and IntelliSense\.

```csharp
public static class EmojiObjects
```
